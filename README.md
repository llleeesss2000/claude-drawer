## ✨ 這是什麼？

Claude 抽屜讓你用圖形介面管理 Claude Code 的一切設定：

- 📦 **技能（Skills）** — 一鍵安裝或移除
- 🔌 **外掛（Extensions）** — 輕鬆啟用或停用
- 🖥️ **MCP 伺服器** — 管理連線、啟動與關閉
- 📄 **全域指令（CLAUDE.md）** — 視覺化編輯

不需要打指令，不需要懂設定檔，雙擊桌面捷徑就能開始。

## 🚀 安裝

確認你已經安裝 [Node.js](https://nodejs.org/) 和 [Claude Code](https://claude.ai/code)，然後執行：

```bash
  curl -fsSL https://raw.githubusercontent.com/llleeesss2000/claude-drawer/main/install.sh
```

安裝完成後，**桌面捷徑會自動建立**，雙擊即可開啟。

> 第一次開啟時，安裝精靈會自動偵測你的環境是否就緒，並引導你完成設定。

## 📖 使用說明

### 第一次使用

1. `npm install -g claude-drawer`
2. 雙擊桌面上的「Claude 抽屜」捷徑
3. 依照安裝精靈完成環境確認
4. 選擇你的語言
5. 告訴我們你主要用 Claude 做什麼，我們幫你推薦最適合的設定

### 日常使用

1. 雙擊桌面捷徑開啟
2. 在「我的設定」頁勾選想要的技能 / 外掛 / MCP
3. 按下「執行變更」
4. 重新啟動 Claude Code

# 🗂️ Claude Drawer

> A graphical manager for Claude Code — no coding required. Manage Skills, Extensions, and MCP servers with ease.

[English](./README.en.md) | [简体中文](./README.zh-CN.md) | [Español](./README.es.md) | 繁體中文

## What is this?

Claude Drawer gives you a graphical interface to manage all your Claude Code settings — Skills, Extensions, MCP Servers, and your global CLAUDE.md — without touching a single config file.

## Install

```bash
npm install -g claude-drawer
```

A desktop shortcut is created automatically. Double-click to launch.
